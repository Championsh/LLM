
public void RemoveAt(int index) {
    if (index >= 0 && index <= _count - 1) {
        int i = 0;
        for (; i <= index; ++i) {
            _items[i] = _items[i + 1];
        }
        --_count;
    } else {
        throw new IndexOutOfRangeException("Index is out of range.");
    }
}


This function takes an integer index as input and removes the element at that index from the _items array. It uses the existing Insert function to shift all elements after the one being removed to fill the gap. If the index is outside the valid range (0 to Count - 1), it throws an IndexOutOfRangeException.
